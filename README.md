Program menampilkan tulisan “PROGRAM MENGHITUNG FAKTORIAL”.
Lalu pengguna diminta memasukkan bilangan bulat positif, yang disimpan dalam variabel n.
long faktorial = 1;
faktorial diisi nilai awal 1 karena dalam operasi perkalian, angka awal harus 1 (bukan 0).
Tipe data long digunakan karena hasil faktorial bisa sangat besar (lebih besar dari int).
if (n < 0) {
    System.out.println("Maaf, bilangan harus positif!");
}
Jika pengguna memasukkan angka negatif, program menampilkan pesan bahwa bilangan harus positif, dan tidak melanjutkan perhitungan.
for digunakan untuk melakukan perulangan dari 1 sampai n.
Pada setiap langkah, nilai faktorial dikalikan dengan nilai i.
Setelah loop selesai, hasil akhir disimpan di variabel faktorial.
